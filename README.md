# Reactistry

Reactistry is a 2D factory-building and automation game focused on real chemical synthesis.

Players solve increasingly complex challenges by creating molecules through chemical reactions.  
The current version primarily focuses on stoichiometry in chemical reactions, requiring players to balance them to produce the correct molecules.

## Installation

The game is available both as a downloadable Windows application and as a web version.

### Web Page

Open the following url in the browser: `https://merotosc.github.io/Reactistry`

### Windows Executable

1. Open the [Releases](https://github.com/merotosc/Reactistry/releases) page.
2. Download the latest Windows release.
3. Run the downloaded `.exe` file.

## Gameplay

After launching the game, the available shortcuts are displayed in the sidebar on the left side.
The tasks that need to be completed are shown on the right side.
The toolbar at the bottom is used to place buildings such as extractors, pipes and reactors on the map.
A quick tutorial can be accessed by clicking the help button in the upper-left corner.
